# Job Market Analysis using SQL

## Project Overview

This project analyzes global data-related job trends using SQL queries in PostgreSQL. The analysis focuses on salaries, experience levels, remote work trends, employment types, and job roles in the data industry.

---

## Tools Used

* PostgreSQL
* SQL
* CSV Dataset

---

## Database Creation

A database named `job_market_analysis` was created and the dataset was imported into a table called `jobs`.

---

## Dataset Columns

* work_year
* experience_level
* employment_type
* job_title
* salary
* salary_currency
* salary_in_usd
* employee_residence
* remote_ratio
* company_location
* company_size

---

## SQL Analysis Performed

### Top 10 Most Common Job Roles

Analyzed the most frequently occurring job titles in the dataset.

### Average Salary by Experience Level

Calculated average salaries for different experience levels.

### Remote Work Distribution

Analyzed the distribution of remote, hybrid, and onsite jobs.

### Highest Paying Roles

Identified the top paying job titles based on average salary in USD.

### Employment Type Distribution

Analyzed different employment types such as full-time and part-time jobs.

---

## Key Insights

* Senior-level positions receive the highest average salaries.
* Full-time jobs dominate the dataset.
* Remote jobs make up a significant portion of data-related roles.
* Data Scientist and Machine Learning roles are among the highest paying jobs.
* Certain job roles appear much more frequently than others.
